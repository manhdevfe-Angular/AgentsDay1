# AgentsDay1 - 3D Tech Portfolio

This is a modern, 3D Personal Landing Page built for an IT professional using vanilla JavaScript, Vite, and Three.js.

## Features
- Interactive 3D Particles Background (Three.js)
- Responsive Modern UI with Glassmorphism
- Neon styling and Smooth gradient animations
- Real profile information and dynamic projects showcase

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
